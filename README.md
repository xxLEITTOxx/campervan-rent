# TravelTrucks - Campervan Rental Platform

Веб-додаток для оренди кемперів від компанії TravelTrucks. Проєкт реалізований з використанням Next.js, TypeScript та Zustand.

## Основні функції

- **Домашня сторінка** - банер з основним закликом до дії
- **Каталог кемперів** - перегляд доступних транспортних засобів з можливістю фільтрації
- **Детальна інформація** - повна інформація про кемпер з галереєю, характеристиками та відгуками
- **Фільтрація** - пошук за локацією, типом транспорту та обладнанням
- **Обране** - збереження улюблених кемперів (зберігається в localStorage)
- **Бронювання** - форма для резервування кемпера
- **Відгуки** - перегляд відгуків від інших користувачів

## Технології

- **Next.js 15.5.4** - React фреймворк з App Router
- **TypeScript** - типізація коду
- **Zustand** - управління глобальним станом
- **Axios** - HTTP клієнт для запитів до API
- **CSS Modules** - стилізація компонентів

## Встановлення та запуск

### Передумови

- Node.js 18+ 
- npm або yarn

### Кроки встановлення

1. Клонуйте репозиторій:
```bash
git clone <repository-url>
cd campervan-rent
```

2. Встановіть залежності:
```bash
npm install
```

3. Створіть файл `.env.local` в корені проєкту (опціонально, API URL вже налаштований в коді):
```env
NEXT_PUBLIC_API_URL=https://66b1f8e71ca8ad33d4f5f63e.mockapi.io
```

**Примітка:** Наразі API URL вже налаштований в коді. Файл `.env.local` потрібен лише якщо ви хочете змінити URL API.

4. Запустіть dev сервер:
```bash
npm run dev
```

5. Відкрийте браузер за адресою [http://localhost:3000](http://localhost:3000)

### Збірка для продакшн

```bash
npm run build
npm start
```

## API

Додаток використовує MockAPI для отримання даних:

- **Base URL**: `https://66b1f8e71ca8ad33d4f5f63e.mockapi.io`
- **Endpoints**:
  - `GET /campers` - отримання списку кемперів (підтримує фільтрацію та пагінацію)
  - `GET /campers/:id` - отримання деталей кемпера

Детальна документація: [MockAPI Documentation](https://github.com/mockapi-io/docs/wiki)

## Маршрутизація

- `/` - Домашня сторінка
- `/catalog` - Каталог кемперів
- `/catalog/:id` - Детальна сторінка кемпера

## Особливості реалізації

- Фільтрація виконується на бекенді (MockAPI)
- Пагінація реалізована через кнопку "Load more"
- Обране зберігається в localStorage через Zustand persist middleware
- Форма бронювання показує нотифікацію про успішну відправку
- Всі асинхронні операції мають індикатор завантаження
